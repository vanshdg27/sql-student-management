# Student Result Management System (SQL Project)

This MySQL-based project manages students, courses, subjects, and their marks. It includes:

- Database schema design
- Sample data inserts
- Queries for average marks, top student, and a view

## 🛠️ Tech Stack
- MySQL

## 🚀 How to Use

1. Run `schema.sql` to create tables.
2. Run `sample_data.sql` to insert initial data.
3. Run `queries.sql` to see key results and the view.
